# 🖥️ MK OS - Browser-Based Desktop Environment

A fully functional **desktop-like operating system UI** built using **HTML, CSS, and Vanilla JavaScript** — no frameworks, no libraries, just pure DOM mastery.

## 📌 Features

### 🖱️ Desktop System
- Draggable desktop icons
- Double-click to open apps
- Custom positioning of icons

### 🪟 Window Manager
- Multiple windows support
- Draggable windows
- Close button functionality
- Dynamic window stacking

### 🧠 Context Menu (Right Click)
- Copy / Paste / Cut / Delete
- Create Folder dynamically

### 📁 File System Simulation
- Create new folders on desktop
- Interactive UI behavior

### 🕒 Live Clock
- Real-time clock with seconds
- Dynamic date display

### ⚙️ Settings Panel
- Sidebar UI (macOS inspired)
- Dark / Light mode toggle
- Brightness control
- Change background wallpaper

### 🎨 UI/UX
- Glassmorphism dock (bottom navbar)
- Smooth hover animations
- Modern desktop feel
